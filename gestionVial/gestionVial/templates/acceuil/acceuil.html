gestionVial

L'application centralisee pour la gestion de la Vial-Mali.

Selectionnez un des modules disponibles dans la barre de navigation a gauche pour afficher les operations possibles reliees a ce module.

Les modules ci-dessous sont disponibles

Stock - pour consulter et/ou mettre a jour le niveau du stock.

03/02/2023 - Fonctionnel

Ventes - pour creer devis et/ou generer une facture pour un devis existant. Vous permet egalement de consulter tous les devis courrants ou passes

Recevables - pour consulter les factures non-payees et effectuer un paiement. Ici vous pourrez egalement consulter tous les factures courrantes ou passees.

Achats - pour effectuer les operations d'achat de stock ou d'equipement. Vous permet de creer des bons de commandes et consulter les factures fournisseur.

Creances - pour consulter les dois, tels que les factures fournisseurs. Vous permet d'effectuer des paiements contres les factures fournisseurs non-payees.

Personnel - pour consulter le repertoire des employes, creer ou modifier le profil des employes. Vous permet aussi de recencer les conges et abscences.
